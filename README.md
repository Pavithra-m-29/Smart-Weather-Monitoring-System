# Smart Weather Monitoring System using ESP8266

## Overview

This project is an IoT-based Smart Weather Monitoring System developed using ESP8266 NodeMCU and DHT11 Sensor. The system measures real-time temperature and humidity and displays the data on a web dashboard accessible through a mobile phone or computer connected to the same Wi-Fi network.

## Features

* Real-time Temperature Monitoring
* Real-time Humidity Monitoring
* Web-Based Dashboard
* Automatic Data Refresh
* Weather Status Display
* Mobile-Friendly Interface

## Components Used

* ESP8266 NodeMCU
* DHT11 Temperature and Humidity Sensor
* LEDs
* Breadboard
* Jumper Wires
* USB Cable

## Working Principle

The DHT11 sensor continuously reads temperature and humidity values from the environment. The ESP8266 processes the sensor data and hosts a local web server. Users can access the dashboard through a browser using the ESP8266 IP address. The webpage automatically refreshes every 5 seconds to display updated values.

## Technologies Used

* Arduino IDE
* ESP8266 WiFi Library
* ESP8266 Web Server
* DHT Sensor Library
* HTML
* CSS
* JavaScript

## Project Output

The system successfully displays real-time weather information, including temperature, humidity, and weather status, on a responsive web dashboard accessible from mobile devices and computers.
